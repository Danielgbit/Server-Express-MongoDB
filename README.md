# Ecommerce con Express, WebSocket y Handlebars

Este proyecto es un ecommerce desarrollado como parte del curso de **Backend** en **Coderhouse**. Utiliza **Express** como framework de servidor, **WebSocket** para comunicación en tiempo real y **Handlebars** como motor de plantillas. El objetivo es crear una plataforma de comercio electrónico funcional y escalable, aplicando los conocimientos adquiridos en el curso.

---

## Características principales

- **Express**: Servidor robusto y rápido para manejar rutas y middleware.
- **WebSocket**: Comunicación en tiempo real entre el servidor y los clientes (ej: notificaciones, actualizaciones de stock).
- **Handlebars**: Motor de plantillas para generar vistas HTML dinámicas.
- **Ecommerce en proceso**: Funcionalidades básicas de un ecommerce, como listado de productos, carrito de compras y autenticación de usuarios.
